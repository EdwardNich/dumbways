// Tipe dan struktur data
JavaScript merupakan bahasa pemrograman yang dinamis. Artinya, variable dalam JavaScript tidak secara langsung dikaitkan dengan tipe data tertentu, dan variabel apa pun dapat ditetapkan (dan ditetapkan kembali) dengan tipe yang berbeda.

Tipe data pada dasarnya menentukan jenis data apa yang dapat disimpan dan dimanipulasi dalam suatu program.  JavaScript memiliki dua tipe data utama, yaitu primitif dan non-primitif.

// Tipe data primitif
Data primitif merupakan tipe data sederhana yang tidak mempunyai method atau properti dan nilainya tidak dapat diubah setelah dibuat / immutable.

JavaScript memiliki 6 tipe data primitif yaitu: string, number, boolean, null, undefined, dan symbol.
